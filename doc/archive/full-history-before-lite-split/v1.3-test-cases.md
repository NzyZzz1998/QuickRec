### 启动前检查

- [ ] 拉取 test 分支最新代码
- [ ] 确认 `ffmpeg/ffmpeg.exe` 存在
- [ ] 在 PyCharm 中运行 `src/main.py` 无报错
- [ ] 系统托盘出现 QuickRec 图标
- [ ] 右键托盘图标显示菜单包含"▶ 全屏录制"、"▢ 区域录制"、"🖥 窗口录制"、"⚙ 设置"、"💻 打开保存文件夹"、"✕ 退出"

### 注意事项

- **不要用 Anaconda 的 Python 运行**，必须用 CPython
- 窗口录制测试（T2.x / T4.x）需要至少一个可见窗口（如记事本、Chrome）
- 磁盘空间测试（T5.x）需注意不要在系统盘根目录测试（避免权限问题）
- H.264 编码验证需要视频播放器（如 VLC / Windows Media Player）

## v1.3 已知限制

1. **仅 Windows 支持**：macOS/Linux 不在 v1.3 范围内
2. **打包体积**：当前优化至 ~259MB，核心依赖（cv2 73MB + ffmpeg 97MB + numpy 27MB + Qt 18MB）约 215MB 不可压缩
3. **窗口丢失处理**：关闭窗口→自动停止保存+托盘通知，最小化→暂停+托盘通知（无弹窗对话框）
4. **磁盘预警**：仅录制前检查一次，不持续监控
5. **DPI 适配**：设置 Qt 高 DPI 属性，大部分 UI 正常缩放，极端缩放比仍有少量偏差可能
