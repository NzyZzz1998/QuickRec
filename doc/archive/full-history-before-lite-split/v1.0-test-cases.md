## v1.0 已知限制

1. **仅全屏录制**：区域录制功能因 Windows 11 兼容性问题推迟到 v1.1
2. **无音频录制**：v1.0 不支持系统声音和麦克风录制
3. **单显示器**：仅录制主显示器
4. **编码格式**：使用 OpenCV mp4v，画质不如 x264
