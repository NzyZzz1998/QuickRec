## 1. 测试环境准备


| 项目     | 要求                                                                                        |
| ------ | ----------------------------------------------------------------------------------------- |
| 操作系统   | Windows 10/11                                                                             |
| Python | CPython 3.12.8（`D:\Work\Software\Python\python.exe`）                                      |
| 依赖     | pynput, dxcam, opencv-python, PyQt5, pystray, pyaudiowpatch, pyaudio, soundcard, winotify |
| FFmpeg | `ffmpeg/ffmpeg.exe` 就位（音频录制必需），或系统 PATH 可用                                                |
| 启动命令   | `D:\Work\Software\Python\python.exe src/main.py`（在项目根目录执行）                                |


### 启动前检查

- [ ] 安装 v1.2 新依赖（无新增外部依赖，均为标准库或已有依赖）
- [ ] 在项目根目录运行 `D:\Work\Software\Python\python.exe src/main.py` 无报错
- [ ] 系统托盘出现 QuickRec 图标
- [ ] 右键托盘图标显示菜单包含"▶ 全屏录制"、"▢ 区域录制"、"🖥 窗口录制"、"⚙ 设置"、"📁 打开保存文件夹"、"✕ 退出"

### 注意事项

- **不要用 Anaconda 的 Python 运行**，必须用 CPython
- 窗口录制测试（T2.x）需要至少一个可见的非系统窗口

## v1.2 已知限制

1. **仅 Windows 支持**：macOS/Linux 不在 v1.2 范围内
2. **单显示器**：窗口录制仅录制主显示器上的窗口
3. **窗口最小化**：最小化的窗口无法录制，自动暂停
4. **鼠标高亮性能**：每次点击约 < 1ms 开销，对帧率无影响
5. **开机自启**：仅写入 HKCU\Run，无需管理员权限，安全软件可能提示
6. **倒计时**：倒计时显示在工具栏内，非全屏遮罩
