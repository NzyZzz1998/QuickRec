# QuickRec Lite v0

Lite v0 是 QuickRec Lite 产品线的第一个轻量化版本，目标是保留稳定的全屏录制和音频录制能力，同时移除 Full 版本中更复杂的区域/窗口录制、鼠标高亮、倒计时和诊断导出能力。

- `prd.md`：Lite v0 需求文档。
- `dev_plan.md`：Lite v0 实施计划。
- `progress.md`：Lite v0 总体进度和最小任务拆分。
- `test-cases.md`：Lite v0 测试用例。
- `package-size-report.md`：Lite v0 打包体积记录。
- `release-notes.md`：Lite v0 发布说明。
- `development-log.md`：Lite v0 开发日志。
