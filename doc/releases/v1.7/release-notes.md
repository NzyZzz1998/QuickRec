# QuickRec Full v1.7 发布说明

> 状态：正式发布
> 收口日期：2026-07-21

## 本版定位

v1.7 聚焦素材发现效率，在 v1.6.1 中央素材库基础上增加关键词搜索、条件筛选和结果排序，不改变索引 schema，也不引入数据库或完整工作台。

## 主要变化

- 支持按文件名或完整路径片段搜索素材。
- 支持按状态、录制模式、音频模式和时间范围筛选。
- 支持按录制时间、时长和文件大小进行六种排序。
- 待入库与正式素材保持独立分区，并分别应用相同查询条件。
- 正式素材按匹配结果每次加载 50 条，待入库记录不分页。
- 同一进程内关闭再打开素材库可恢复查询条件；应用重启后恢复默认状态。
- 查询异常保留上一次有效结果，日志不记录查询关键词和素材路径。

## 明确不包含

- 不包含素材预览、收藏、标签、项目分类和完整工作台。
- 不包含 AI、剪辑、导出队列、云同步、WGC 或 120 FPS。
- 不修改 QuickRec Lite。
