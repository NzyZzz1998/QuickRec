# QuickRec Full v1.7 变更日志

> 状态：正式发布
> 收口日期：2026-07-21

## 新增

- 素材文件名与完整路径片段搜索。
- 素材状态、录制模式、音频模式和录制时间筛选。
- 录制时间、时长和文件大小六种排序。
- 待入库与正式素材独立分区、独立计数和一致查询条件。
- 正式素材每次加载 50 条，待入库记录不分页。
- 同进程查询条件保持，应用重启恢复默认条件。

## 改进

- 查询异常时保留上一次有效结果，并显示非阻断反馈。
- 索引加载失败时禁用不安全查询操作，同时保留可恢复入口。
- 素材库打开期间，新录制完成后自动刷新列表。
- 文本输入获得焦点时，不再由全局 `Ctrl+A` 触发区域录制。
- 查询日志不记录关键词、素材文件名、路径或具体条件。
