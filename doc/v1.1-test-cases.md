## v1.1 已知限制

1. **仅 Windows 支持**：macOS/Linux 不在 v1.1 范围内
2. **单显示器**：仅录制主显示器
3. **指定窗口录制**：推迟到 v2.0
4. **WASAPI 兼容性**：部分声卡可能不支持环形缓冲区，自动降级为无声录制
5. **FFmpeg 打包**：需手动下载 ffmpeg.exe 到 ffmpeg/ 目录
