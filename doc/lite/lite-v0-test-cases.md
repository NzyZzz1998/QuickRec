# QuickRec Lite v0 测试用例

> 版本: QuickRec Lite v0  
> 创建时间: 2026-07-08  
> 测试环境: Windows 10/11, Python 3.12.x, PyQt5  
> 基线: QuickRec v1.4 稳定版本  
> 测试方式: 自动化测试 + 硬件冒烟 + Computer Use 可视化验证 + 手动音频验证

---

## 1. 测试环境准备

| 项目 | 要求 |
| --- | --- |
| 操作系统 | Windows 10/11 |
| Python | CPython 3.12.x |
| 分支 | `lite-test` |
| 依赖 | `python -m pip install -r requirements-dev.txt` |
| FFmpeg | `ffmpeg/ffmpeg.exe` 存在，或打包产物内置 FFmpeg |
| 输出目录 | 建议使用 `E:\QRtest` |
| 开发运行 | 项目根目录作为 working directory 运行 `src/main.py` |

### 启动前检查

- [ ] 当前分支为 `lite-test`。
- [ ] `python -m compileall src scripts tests` 通过。
- [ ] `python -m ruff check .` 通过。
- [ ] `python -m mypy` 通过。
- [ ] `python -m pytest -q` 通过。
- [ ] `python scripts\hardware_smoke.py --output-dir E:\QRtest --duration 3 --mode fullscreen` 通过。
- [ ] 若验证音频，确认系统声音输出设备与麦克风可用。

### 注意事项

- Lite v0 只保留全屏录制作为用户可见录制模式。
- 区域录制、窗口录制、录制倒计时和鼠标点击高亮在 Lite v0 中不应出现入口。
- 旧配置文件中即使存在 `quality=high/medium/low`、`fps=30`、`show_countdown=true`、`mouse_highlight=true`，Lite v0 实际录制仍应保持原生分辨率、60fps、无倒计时、无鼠标点击高亮。
- 打包体积低于 200MB 是目标，不作为 Lite v0 阻断验收项。
