### 4.2 注册表查看方法

测试 T4.1-T4.5 需要检查注册表，可通过以下方式验证：

**方法一：Windows 注册表编辑器（推荐）**

1. 按 `Win+R`，输入 `regedit`，回车
2. 导航到 `HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run`
3. 右侧列表中查找名为 `QuickRec` 的项
  - 存在：双击可查看值（应指向 Python 解释器或 exe 路径）
  - 不存在：说明开机自启未开启或已关闭

**方法二：命令行查询**

```cmd
reg query "HKCU\Software\Microsoft\Windows\CurrentVersion\Run" /v QuickRec
```

- 有值时输出类似：`QuickRec  REG_SZ  D:\Work\Software\Python\python.exe`
- 未开启时输出：`错误: 系统找不到指定的文件或注册表项`

**方法三：Python 脚本验证**

```cmd
D:\Work\Software\Python\python.exe -c "import winreg; key=winreg.OpenKey(winreg.HKEY_CURRENT_USER,r'Software\Microsoft\Windows\CurrentVersion\Run',0,winreg.KEY_READ); print(winreg.QueryValueEx(key,'QuickRec')); winreg.CloseKey(key)"
```

- 输出类似：`('D:\\Work\\Software\\Python\\python.exe', 1)`
- 未开启时抛出 `FileNotFoundError`

**注意事项：**

- 开发环境下注册表值指向 `D:\Work\Software\Python\python.exe`（正常现象）
- 打包后值指向 `QuickRec.exe` 所在路径
- 操作 HKCU\Run 无需管理员权限
- 安全软件可能提示注册表修改，选择允许即可
