# 任务：系统托盘模块 (tray_icon.py)

**模块**：`src/ui/tray_icon.py`
**说明**：管理系统托盘图标。

## 前置依赖
- [ ] 安装 pystray 包 (pip install pystray)
- [x] PyQt5 5.15.10 (Anaconda 已安装)

## 子任务

### 1. 实现 TrayIcon 类
- [ ] 使用 pystray 创建托盘图标
- [ ] 默认图标 (使用自定义图标或系统图标)

### 2. 菜单项
- [ ] "▶ 开始录制" → 触发录制
- [ ] "⚙ 设置" → 打开设置对话框
- [ ] "📁 打开保存文件夹" → 打开 save_path
- [ ] "───────────" (分隔线)
- [ ] "✕ 退出" → 退出程序

### 3. 功能
- [ ] `show()` 显示托盘图标
- [ ] `hide()` 隐藏托盘图标
- [ ] `show_notification(msg)` 弹出系统通知
- [ ] `set_menu(menu_items)` 设置菜单

## 验收标准
- [ ] 托盘图标正常显示
- [ ] 菜单项点击触发正确
- [ ] 通知弹出正常
